To find the wavelength of a standing wave on a string, we need to look at the relationship between the number of antinodes and the length of the string.

### 1. Identify the Mode of Vibration
For a string fixed at both ends, each **antinode** corresponds to one "loop" or "half-wavelength." 
* A standing wave with **four antinodes** is known as the **fourth harmonic** ($n = 4$).



### 2. Relate Length to Wavelength
Since each antinode represents half a wavelength ($\frac{\lambda}{2}$), a string with four antinodes contains exactly two full wavelengths:
$$L = 4 \times \frac{\lambda}{2} = 2\lambda$$

Or, using the general formula for a string fixed at both ends:
$$\lambda_n = \frac{2L}{n}$$
Where:
* $L = 80 \text{ cm} = 0.8 \text{ m}$
* $n = 4$ (number of antinodes)

### 3. Calculation
Substitute the values into the formula:
$$\lambda = \frac{2 \times 0.8 \text{ m}}{4}$$
$$\lambda = \frac{1.6 \text{ m}}{4}$$
$$\lambda = 0.4 \text{ m}$$

---

**Final Answer:**
The wavelength of this wave is **$0.4 \text{ m}$** (or **$40 \text{ cm}$**).
