To identify the local maxima or minima of the function $f(x) = 3x^2 - 12x + 7$, we can use calculus or complete the square. Since this is a quadratic function with a positive leading coefficient ($a=3$), we know it represents a parabola that opens **upward**, meaning it will have a **local minimum** at its vertex.

### 1. Find the Derivative
The first derivative $f'(x)$ gives the slope of the tangent line. Local extrema occur where the slope is zero.
$$f'(x) = \frac{d}{dx}(3x^2 - 12x + 7)$$
$$f'(x) = 6x - 12$$

### 2. Solve for Critical Points
Set the derivative to zero to find the x-coordinate of the vertex:
$$6x - 12 = 0$$
$$6x = 12$$
$$x = 2$$



### 3. Determine the y-coordinate
Substitute $x = 2$ back into the original function to find the value of the minimum:
$$f(2) = 3(2)^2 - 12(2) + 7$$
$$f(2) = 3(4) - 24 + 7$$
$$f(2) = 12 - 24 + 7$$
$$f(2) = -5$$

---

### 4. Verification (Second Derivative Test)
To confirm if it is a maximum or minimum, we look at the second derivative:
$$f''(x) = 6$$
Since $f''(x) > 0$, the function is **concave up** at the critical point, confirming that $x=2$ is a **local minimum**.

### Summary
* **Type:** Local Minimum
* **Coordinates:** $(2, -5)$
* **Local Maxima:** None (the function increases to infinity as $x \to \pm\infty$)
